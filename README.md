## 简介
worklog用于团队记录工作的日志，每天、每周（可根据每天自动生成）、请假等   

后端：Nodejs、Koa2、art-template    

前端：vue2、element-ui   

数据库：mysql  

有问题请联系 [hu_xi_yu@163.com](mailto:hu_xi_yu@163.com).  

## 快速开始  
下载和运行:

```
git clone https://github.com/huxiyu/worklog
cd worklog
npm install
npm start
```

浏览器打开 http://localhost:8080 ，不支持IE8及以下。
